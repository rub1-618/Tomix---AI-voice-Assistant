1. Відкрий сайт: https://aistudio.google.com

2. Увійди у свій Google аккаунт.

3. У лівому меню натисни "Get API key" → "Create API key".

4. Скопіюй ключ (починається з "AIza...").

5. Встав його у поле "Gemini API Key" у налаштуваннях Tomix
   і натисни "Зберегти".

БЕЗКОШТОВНІ ЛІМІТИ (станом на 2025-2026):
  • Gemini 2.0 Flash   — 15 запитів/хв, 1 500/день — БЕЗКОШТОВНО
  • Gemini 2.5 Pro     — 50 запитів/день — БЕЗКОШТОВНО

Для звичайного використання Tomix цього достатньо.
